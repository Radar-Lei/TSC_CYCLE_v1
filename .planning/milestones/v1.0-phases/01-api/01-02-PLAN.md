    模块 docstring: "从 train.jsonl 分层抽样训练数据，确保交叉口和饱和度分布均衡" (中文)

    import: `json, random, os, collections` + `from typing import List, Dict, Tuple, Optional` + `from dataclasses import dataclass, field`

    **饱和度分桶函数** `_saturation_bucket(sample_dict: dict) -> str`:
    - 计算 sample 中所有 phase_waits 的 avg pred_saturation
    - `< 0.3` 返回 `"low"`, `0.3-0.7` 返回 `"med"`, `>= 0.7` 返回 `"high"`

    **`@dataclass class SamplingStats`**:
    - `total_source: int`
    - `total_sampled: int`
    - `tl_id_counts: Dict[str, int]` (每个 tl_id 的采样数)
    - `bucket_counts: Dict[str, int]` (low/med/high 各多少)
    - `to_dict()` 方法
    - `print_summary(self)` 方法: 打印 `[采样] 总量: X/Y`, `[采样] 饱和度分布: low=X, med=Y, high=Z`, `[采样] 交叉口覆盖: X/Y`

    **`class StratifiedSampler`**:
    - `__init__(self, seed: int = 42)`: 保存 seed

    - `load_data(self, jsonl_path: str) -> List[dict]`:
      - 逐行 `json.loads()` 读取 jsonl 文件
      - 返回 `List[dict]` (原始 dict, 不转 dataclass, 节省内存)
      - 打印 `[采样] 加载 {n} 条数据`

    - `sample(self, data: List[dict], n: int = 5000) -> Tuple[List[dict], SamplingStats]`:
      - 分层策略:
        1. 按 `(tl_id, bucket)` 组合分组: `groups = defaultdict(list)`，key 为 `(metadata["tl_id"], _saturation_bucket(item))`
        2. 计算每组配额: `quota = max(1, round(n * len(group) / len(data)))`，确保每组至少 1 个
        3. 如果某组样本数 < quota，取全部
        4. 对每组 `random.Random(self.seed).sample(group, quota)`
        5. 如果总数 > n，随机去掉多余的；如果 < n，从未选中的样本中随机补充
      - 返回 `(sampled_list, SamplingStats)`

    **模块级便捷函数** `sample_training_data(input_path: str, n: int = 5000, output_path: Optional[str] = None, seed: int = 42) -> Tuple[List[dict], SamplingStats]`:
    - 创建 StratifiedSampler(seed)
    - 加载数据、执行抽样
    - 如果 output_path 不为 None，将结果逐行写入 jsonl (一行一个 json.dumps)
    - 调用 stats.print_summary()
    - 返回 (sampled, stats)

    **CLI 入口** `if __name__ == "__main__":`:

    - 调用 `sample_training_data()`
