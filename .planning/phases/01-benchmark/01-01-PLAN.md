  <behavior>
    加权平均计算：
    - 输入：List[CycleResult] + 对应周期时长列表
    - 输出：按周期时长加权的统计平均值
    - 公式：weighted_avg = sum(value_i * duration_i) / sum(duration_i)
    - 零值处理：周期时长为 0 的场景跳过不参与计算

    Throughput 计算：
    - 每周期 throughput = passed_vehicles / cycle_duration
    - 汇总 throughput = 各周期 throughput 按周期时长加权平均

    统计指标加权：
    - queue_vehicles_avg: 按周期数加权
    - total_delay_avg: 按周期数加权
    - throughput: 按周期时长加权
  </behavior>

       ```python
       def calculate_weighted_average(values: list[float], weights: list[float]) -> float:
           """按权重计算加权平均值，跳过零权重"""
           filtered = [(v, w) for v, w in zip(values, weights) if w > 0]
           if not filtered:
               return 0.0
           total_weight = sum(w for _, w in filtered)
           if total_weight == 0:
               return 0.0
           return sum(v * w for v, w in filtered) / total_weight
       ```
