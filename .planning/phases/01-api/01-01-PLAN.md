    2. 创建 `src/glm5/client.py`:
       - 模块 docstring: "GLM-5 API 客户端，支持并发请求和指数退避重试" (中文)
       - `import os, time, logging` + `from concurrent.futures import ThreadPoolExecutor, as_completed` + `from dataclasses import dataclass` + `from typing import Optional, List` + `import openai`

       - `@dataclass class GLM5Response`:
         - `content: str`
         - `response_time: float`
         - `success: bool`
         - `error: Optional[str] = None`
         - `to_dict()` 和 `from_dict()` 方法 (遵循项目惯例)

       - `class GLM5Client`:
         - `__init__(self, api_key: Optional[str] = None, max_retries: int = 3, retry_base_delay: float = 2.0, timeout: float = 120.0, max_concurrent: int = 4)`:
           - `api_key` 优先使用参数，否则 `os.environ.get("GLM_API_KEY")`，都没有则 `raise ValueError("GLM_API_KEY not set")`
           - `self._client = openai.OpenAI(base_url="https://open.bigmodel.cn/api/coding/paas/v4", api_key=api_key)`
           - `self.model = "glm-5"`
           - `self.max_tokens = 8192` (per API-04, 硬编码不可覆盖)
           - `self.max_retries = max_retries`
           - `self.retry_base_delay = retry_base_delay`
           - `self.timeout = timeout`
           - `self.max_concurrent = max_concurrent`
           - `self._executor = ThreadPoolExecutor(max_workers=max_concurrent)`

         - `call_single(self, system_prompt: str, user_prompt: str) -> GLM5Response`:
           - 重试循环 `for attempt in range(self.max_retries + 1)`:
             - `self._client.chat.completions.create(model=self.model, messages=[{"role":"system","content":system_prompt},{"role":"user","content":user_prompt}], max_tokens=self.max_tokens, timeout=self.timeout)`
             - 成功则返回 `GLM5Response(content=..., response_time=..., success=True)`
             - 捕获 `openai.APITimeoutError`, `openai.APIConnectionError`, `openai.RateLimitError`, `openai.APIError` 异常
             - 退避延迟: `delay = self.retry_base_delay * (2 ** attempt)` (2s, 4s, 8s)
             - 日志: `print(f"[GLM5] 重试 {attempt+1}/{self.max_retries}: {error}")` (项目风格)
           - 所有重试失败: 返回 `GLM5Response(content="", response_time=total, success=False, error=last_error)`

         - `call_batch(self, requests: List[dict]) -> List[GLM5Response]`:
           - `requests` 格式: `[{"system_prompt": str, "user_prompt": str}, ...]`
           - 使用 `self._executor` 提交 `call_single` 任务
           - `futures = {self._executor.submit(self.call_single, r["system_prompt"], r["user_prompt"]): i for i, r in enumerate(requests)}`
           - 收集结果保持原始顺序
           - 返回 `List[GLM5Response]`

         - `shutdown(self)`: 关闭 `self._executor`
