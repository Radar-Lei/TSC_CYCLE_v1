    **脚本实现要点:**

    1. **命令行接口**: argparse，参数:
       - `--input`: 输入文件路径（默认 `outputs/data/train.jsonl`）
       - `--output`: 输出文件路径（默认 `outputs/grpo/grpo_train.jsonl`）
       - `--config`: 配置文件路径（默认 `config/config.json`，预留扩展）

    2. **System prompt**: 直接定义为常量，内容与 Task 1 更新后的 SYSTEM_PROMPT 一致:
       ```python
       SYSTEM_PROMPT = (
           "你是交通信号配时优化专家。\n"
           "请认真分析预测得到的下个周期各个相位的交通状态，给出下个周期的配时方案，并给出你的推理过程。\n"
           "将推理过程放在 <think> 和 </think> 之间。\n"
           "然后，将你的最终方案放在 <CyclePlan> 和 </CyclePlan> 之间。"
       )
       ```
       注意措辞强调"分析"和"推理过程"（per 用户决策: 强调"优化"而非"学习格式"）。

    3. **User content 提取**:
       从 train.jsonl 每条样本的 `prompt` 字段中，去掉第一行（旧的 system prompt "你是交通信号配时优化专家。"），剩余部分作为 user content。
       代码逻辑:
       ```python
       lines = sample['prompt'].split('\n')
       user_content = '\n'.join(lines[1:])  # 去掉第一行 system prompt
       ```

    4. **State file 路径转换**:
       将 `state_file` 从绝对路径转换为相对路径:
       ```python
       # 原始: "/home/samuel/SCU_TSC/outputs/states/arterial4x4_10/state_xxx.xml"
       # 目标: "outputs/states/arterial4x4_10/state_xxx.xml"
       state_file = sample['state_file']
       # 查找 "outputs/states/" 在路径中的位置并截取
       marker = "outputs/states/"
       idx = state_file.find(marker)
       if idx != -1:
           relative_state_file = state_file[idx:]
       else:
           relative_state_file = state_file  # fallback
       ```

    5. **输出格式** (每行一个 JSON 对象):
       ```json
       {
         "prompt": [
           {"role": "system", "content": "<SYSTEM_PROMPT>"},
           {"role": "user", "content": "<user_content>"}
         ],
         "metadata": {
           "state_file": "outputs/states/arterial4x4_10/state_xxx.xml",
           "tl_id": "nt7",
           "sim_time": 350.0,
           "date": "2026-01-02",
           "cycle_count": 6
         }
       }
       ```
       注意: **没有 answer 字段**（per 用户决策: reward 由 SUMO 仿真计算）。
       metadata 合并: 将原始 metadata（tl_id, sim_time, date, cycle_count）加上 state_file 放入 metadata 字段。

    6. **统计输出**: 打印总样本数、各场景分布、state_file 路径转换结果。

    7. **运行脚本生成数据**:
       ```bash
       mkdir -p outputs/grpo
       python3 -m src.scripts.generate_grpo_data --input outputs/data/train.jsonl --output outputs/grpo/grpo_train.jsonl
       ```
