  <action>
创建 src/sft/ 目录和训练脚本 src/sft/train.py，参考 qwen3_(4b)_grpo.py 中 SFT 部分的做法。

1. **加载配置：** 从 config/config.json 读取所有超参数（training.sft 部分）和路径（paths 部分）
   - 支持 --config 参数指定配置文件路径

2. **模型加载（参考 qwen3_(4b)_grpo.py 第 9-33 行）：**
   ```python
   from unsloth import FastLanguageModel
   model, tokenizer = FastLanguageModel.from_pretrained(
       model_name=config["model"]["model_name"],  # "unsloth/Qwen3-4B-Base"
       max_seq_length=config["model"]["max_seq_length"],  # 2048
       load_in_4bit=config["model"]["load_in_4bit"],  # False
       fast_inference=True,
       max_lora_rank=config["model"]["lora_rank"],  # 32
       gpu_memory_utilization=config["model"]["gpu_memory_utilization"],  # 0.9
   )
   model = FastLanguageModel.get_peft_model(
       model,
       r=config["model"]["lora_rank"],
       target_modules=config["model"]["target_modules"],
       lora_alpha=config["model"]["lora_alpha"],
       use_gradient_checkpointing="unsloth",
       random_state=config.get("seed", 3407),
   )
   ```

3. **Chat Template（参考 qwen3_(4b)_grpo.py 第 41-81 行，但适配我们的标签）：**
   - 我们的标签：`<think>` (reasoning_start), `<think>` (reasoning_end, 用相同标签作为关闭), `<solution>` (solution_start), `<solution>` (solution_end, 同理)
   - 注意：此项目使用 `<think>...<think>` 和 `<solution>...<solution>` 格式（开标签重复作为关闭标签）
   - system_prompt = "你是交通信号配时优化专家。"
   - chat template 结构参考 qwen3_(4b)_grpo.py，但替换标签为我们的格式：
     - reasoning_start = "<think>"
     - reasoning_end = "<think>"  (注意：与 start 相同，这是项目约定)
     - solution_start = "<solution>"
     - solution_end = "<solution>"  (同理)
   - add_generation_prompt 应该在最后添加 `<think>` 以引导模型开始思考

4. **数据加载：**
   - 读取 outputs/sft/sft_train.jsonl
   - 每行包含 {"messages": [{role, content}, ...]}
   - 使用 tokenizer.apply_chat_template() 将 messages 转为 text
   - 过滤掉超过 max_seq_length/2 的样本（参考 qwen3_(4b)_grpo.py 第 146-149 行）
   - 转为 HuggingFace Dataset

5. **训练（参考 qwen3_(4b)_grpo.py 第 161-182 行）：**
   ```python
   from trl import SFTTrainer, SFTConfig
   trainer = SFTTrainer(
       model=model,
       tokenizer=tokenizer,
       train_dataset=dataset,
       args=SFTConfig(
           dataset_text_field="text",
           per_device_train_batch_size=config["per_device_train_batch_size"],
           # ... 其余参数从 config 读取
       ),
   )
   trainer.train()
   ```

6. **保存模型（per user decision: 合并 LoRA 保存完整模型）：**
   ```python
   model.save_pretrained_merged(
       config_paths["sft_output"],  # "outputs/sft/model"
       tokenizer,
       save_method="merged_16bit",
   )
   ```

7. 创建空的 src/sft/__init__.py
