       ```python
       class BatchGenerator:
           def __init__(self, client: GLM5Client, output_path: str = "outputs/glm5/results.jsonl", max_retries: int = 3):
               self.client = client
               self.output_path = output_path
               self.max_retries = max_retries  # 约束违反重试次数 (per GEN-05)
               self._completed_ids: set[str] = set()
               self._stats = {"success": 0, "failed": 0, "skipped": 0, "total_think_len": 0}
       ```

    3. **断点续传** (per PROG-01, PROG-03):
       ```python
       def _load_completed(self):
           """启动时读 output_path 已有结果，收集已完成 ID"""
           if not os.path.exists(self.output_path):
               return
           with open(self.output_path, "r") as f:
               for line in f:
                   if line.strip():
                       record = json.loads(line)
                       self._completed_ids.add(record["id"])
                       # 恢复统计
                       if record["status"] == "success":
                           self._stats["success"] += 1
                           self._stats["total_think_len"] += record.get("think_length", 0)
                       else:
                           self._stats["failed"] += 1
           if self._completed_ids:
               print(f"[生成] 断点续传: 已完成 {len(self._completed_ids)} 条，跳过")
       ```

    4. **单条处理** (per GEN-05, GEN-06):
       ```python
       def _process_one(self, sample: dict) -> dict:
           """处理单条样本: 调用 GLM-5 -> 解析 -> 校验 -> 重试"""
           sample_id = _sample_id(sample)
           system_prompt, user_prompt = build_glm5_prompts(sample)
           phase_waits = sample["prediction"]["phase_waits"]

           for attempt in range(self.max_retries):
               response = self.client.call_single(system_prompt, user_prompt)
               if not response.success:
                   # API 错误 (client 已重试过)，直接标记失败
                   return {"id": sample_id, "status": "api_error", "error": response.error, ...}

               parsed = parse_glm5_output(response.content)
               if not parsed.success:
                   continue  # 解析失败，重试

               valid, reason = validate_constraints(parsed.solution, phase_waits)
               if valid:
                   return {"id": sample_id, "status": "success", "think_text": parsed.think_text,
                           "solution": parsed.solution, "think_length": parsed.think_length,
                           "response_time": response.response_time, "retries": attempt,
                           "sample": sample}

               # 约束违反，重试 (per GEN-05)

           # 3 次均失败 (per GEN-06)
           print(f"[生成] 跳过 {sample_id}: {self.max_retries} 次均约束违反")
           return {"id": sample_id, "status": "constraint_failed", "error": reason, "retries": self.max_retries, "sample": sample}
       ```

    5. **进度显示** (per PROG-02):
       ```python
       def _print_progress(self, current: int, total: int):
           """单行刷新进度"""
           pct = current / total * 100
           success = self._stats["success"]
           failed = self._stats["failed"]
           total_done = success + failed
           success_rate = (success / total_done * 100) if total_done > 0 else 0
           avg_think = (self._stats["total_think_len"] / success) if success > 0 else 0
           print(f"\r[生成] {current}/{total} ({pct:.1f}%) | 成功率: {success_rate:.1f}% | 平均think: {avg_think:.0f} tok", end="", flush=True)
       ```

    6. **主循环** — generate(samples: list[dict]):
       - 调用 self._load_completed() 加载断点
       - 创建 output_path 的父目录 (os.makedirs)
       - 使用 ThreadPoolExecutor(max_workers=client.max_concurrent) 并发处理
         - 从 samples 中过滤掉 _completed_ids 中已有的
         - 每个 future 完成后: 立即将结果 json.dumps + "\n" 追加写入 output_path (per PROG-03)
         - 更新 _stats 并调用 _print_progress
       - 结束时打印最终统计摘要
       - 返回统计 dict

    7. 注意事项:
       - 文件写入使用 "a" 模式追加，每条写入后 flush (per PROG-03)
       - ThreadPoolExecutor 使用 client.max_concurrent (默认 4) 作为 max_workers
       - 不要在 generator 里创建 GLM5Client，由外部传入 (依赖注入)
       - 使用 `from concurrent.futures import ThreadPoolExecutor, as_completed` (项目已有模式)
       - ensure_ascii=False 序列化 JSON (项目惯例)
